    15 ♥ - CSS units - Đơn vị CSS
    Công dụng: Tùy chỉnh kích thước trên web
    - Absolute units: Đvị tuyệt đối
        1px ~ 1 điểm ảnh thực trên màn hình
        => Sở hữu kích thước cố định
        EX: px
    - Relative units: Đvị tương đối
        => Sở hữu kích thước không cố định, nó phụ thuộc vào đối tượng thay đổi
        EX: % - phụ thuộc vào thể chứa nó, rem - phụ thuộc vào thẻ html, em - phụ thuộc vào thẻ gần nhất chứa nó
        - vw : viewport wight - Chiếm hết chiều ngang trình duyệt
        - vh : viewport hight - Chiếm hết chiều dọc trình duyệt
